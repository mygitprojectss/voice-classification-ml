# 🎤 Voice Gender Classification using Machine Learning

This project predicts whether a voice belongs to **Male or Female** using machine learning and audio feature extraction.

The system analyzes acoustic properties such as pitch, spectral features, and energy from voice recordings.

---

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🧩 Voice Clustering Visualization
- 🔮 Voice Gender Prediction
- 🎧 Audio Upload Prediction
- 📉 Model Performance Evaluation
- 📈 Audio Signal Visualization

---

## ⚙️ Technologies Used

| Category | Technology |
|--------|-------------|
| Programming | Python |
| Machine Learning | Scikit-Learn |
| Audio Processing | Librosa |
| Data Analysis | Pandas, NumPy |
| Visualization | Matplotlib, Seaborn |
| Web Framework | Streamlit |

---

## 🧠 Machine Learning Models

The project includes:

- **Classification Model** – Predict voice gender
- **Clustering Model** – Identify voice patterns
- **StandardScaler** – Feature scaling
